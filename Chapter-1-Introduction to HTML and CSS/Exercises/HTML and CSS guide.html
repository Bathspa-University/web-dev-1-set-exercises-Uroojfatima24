HTML & CSS Quick Reference Guide

A handy reference for common HTML tags and CSS properties

HTML Tags

  1. <div>:

    Defines a division or a section in an HTML document. 
    <div class="container"> 
    <p>This is a paragraph inside a div</p> 
</div>

  2. <p>:

    Defines a paragraph. 
    <p>This is a paragraph</p>

  3. <ul>:

    Defines an unordered list. 
    <ul>
    <li>Item 1</li>
    <li>Item 2</li>
</ul>

  4. <ol>:

    Defines an ordered list. 
    <ol>
    <li>First item</li>
    <li>Second item</li>
</ol>

  5. <img>:

    Defines an image. 
    <img src="image.jpg" alt="Description">

  6. <a>:

    Defines a hyperlink. 
    <a href="https://example.com">Link</a>

  7. <form>:

    Defines an HTML form for user input. 
    <form action="/submit" method="post">
    <label for="username">Username:</label>
    <input type="text" id="username" name="username">
</form>

  8. <table>:

    Defines a table. 
    <table>
    <tr>
        <td>Row 1, Column 1</td>
        <td>Row 1, Column 2</td>
    </tr>
</table>

  9. <header>:

    Defines a header for a document or section. 
    <header>
    <h1>Page Title</h1>
</header>

  10. <footer>:

    Defines a footer for a document or section. 
    <footer>
    <p>Copyright © 2024 Company Name</p>
</footer>

CSS Properties

  1. color:

    Sets the color of text. 
    body {
    color: #333;
}

  2. font-family:

    Sets the font family for text. 
    p {
    font-family: Arial, sans-serif;
}

  3. background-color:

    Sets the background color of an element. 
    header {
    background-color: #f2f2f2;
}

  4. padding:

    Sets the padding space inside an element. 
    .container {
    padding: 20px;
}

  5. margin:

    Sets the margin space outside an element. 
    img {
    margin: 10px;
}

  6. border:

    Sets the border properties of an element. 
    table {
    border: 1px solid #ddd;
}

  7. text-align:

    Sets the horizontal alignment of text. 
    footer {
    text-align: center;
}

  8. display:

    Specifies the type of rendering box used for an element. 
    ul {
    display: flex;
}
